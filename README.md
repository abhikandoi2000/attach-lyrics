attach-lyrics
=============

Fetch lyrics corresponding to each music file in the given directory


Dependencies
=============

  tagpy 2013.1
  mutagen v1.20


Install Dependencies
=============

Install tagpy

  pip install tagpy

Install mutagen

  pip install mutagen


Usage
=============

  python attach-lyrics.py <path-to-folder>

Example
  
  python attach-lyrics.py "/home/user/songs"
